OOP in JavaScript

Hello !

OOP is a programming paradigm based on the concept of “objects” — which can contain data (properties) and behavior (methods). It's like modeling real-world entities in code — making your programs more organized, reusable, and easier to manage.

OOP in JS Can Be Done In Two Ways:
1. Prototype-based (older style)
2. Class-based (modern ES6+ style) ← most common now
